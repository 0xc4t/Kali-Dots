## Requirements
- i3-wm-gaps
- picom (https://github.com/jonaburg/picom)
- alacritty
- polybar
- cava
- dmenu
- feh
- font - JetBrainMono (https://www.jetbrains.com/lp/mono/)

## Command For Kali
```
sudo apt install kali-desktop-i3-gaps kali-desktop-i3 alacritty polybar cava dmenu feh
```
